== Description ==
Ένα custom WordPress plugin για τη διαχείριση εσόδων και εξόδων, σύμφωνα με τις προδιαγραφές του Financial Management System.

== Installation ==
1. Ανεβάστε τον φάκελο `my-finance` στον κατάλογο `/wp-content/plugins/`.
2. Ενεργοποιήστε το plugin από το μενού 'Plugins' στο WordPress.
