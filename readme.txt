== Description ==
A professional WordPress plugin for financial management. Track income, expenses, and monthly budgets with interactive charts and REST API integration.

== Installation ==
1. Upload the `my-finance-plugin` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
